# Checkbox

You can use simple icons
<Canvas>
  <Story name="Icon">
    <Icon name="close"/>
  </Story>
</Canvas>

You can color them using the `text-{color}` class

<Canvas>
  <Icon name="close" className="text-plum-40"/>
</Canvas>

You can make them clickable by passing `onClick`

<Canvas>
  <Icon name="close" onClick={() => {
    //noop
  }}/>
</Canvas>
